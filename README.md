Порядок запуска:
1. Скачайте проект и откройте папку в PyCharm
2. В первом терминале выполните:
python main.py
Напишите 1 и нажмите Enter (запуск сервера)
3. Откройте второй терминал, снова выполните:
python main.py
Напишите 2 и нажмите Enter (запуск клиента)
